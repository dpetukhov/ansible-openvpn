Пет-проект по созданию своего openvpn/wg сервера.

Целью проекта является:
- научиться создавать приватную сеть
- научиться делать роутинг
- улучшить понимание сетей
- сделать удобный сервис для себя и своих друзей


## Установка сервера

```
ansible-playbook -i inventory.ini playbooks/install-openvpn.yaml
```
